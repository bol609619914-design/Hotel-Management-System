# 更新日志

## 2026-04-23

### 新增

- 新增房态日历，可按日期查看每个房间的空房、已预订、已入住、维修停用状态。
- 房态日历补充颜色图例和悬浮详情，展示房间、日期、订单号、住客和清洁状态。
- 新增入住续住功能，支持前台为未完成订单延长离店日期，并自动重算房费与订单合计。
- 新增换房功能，支持前台为订单更换目标房间，并校验目标房间状态和日期冲突。
- 新增财务流水模块，记录房费、早餐、加床、押金、优惠券、入住确认、退房结算、续住调整和换房调整。
- 新增财务汇总卡片，展示流水笔数、入账金额、优惠抵扣、退款金额和净流水。
- 新增操作日志模块，记录创建订单、更新订单、删除订单、状态变更、续住、换房等关键动作。
- 操作日志页面完成中文化展示，避免直接暴露 `EXTEND_STAY`、`BOOKED`、`reservationNo=...` 等技术字段。
- 新增消息提醒中心，支持预订成功、即将入住、即将退房提醒。
- 首页新增未读提醒卡片，顶部提醒入口新增未读角标。
- 新增酒店运营扩展接口：房态日历、财务流水、财务汇总、操作日志、消息提醒、提醒已读。
- 新增数据库迁移脚本 `2026-04-23_hotel_ops_extension.sql`，包含财务流水、操作日志和消息提醒三张表。

### 优化

- 订单页、财务流水页、日志页、提醒页中的状态、渠道、角色、动作类型统一改为中文展示。
- README 补充房态、续住、换房、流水、日志、提醒等新增业务说明。
- 前端首页进一步强化日常工作台视角，补充提醒与流水概览。

### 修复

- 修复订单号按秒生成导致高频创建订单时可能重复的问题，改为毫秒时间戳加纳秒后缀。
- 修复住客档案在同身份证更换手机号时手机号不同步的问题。
- 增加手机号与身份证归属不同住客时的冲突校验，避免错误合并档案。

### 验证

- 后端编译通过：`mvn -DskipTests compile`。
- 前端构建通过：`pnpm build`。
- 完成 47 项 HTTP 冒烟测试，覆盖登录、权限、房型、房间、订单、入住、离店、续住、换房、取消、住客、房态、流水、日志、提醒、Excel 导出和住客端预订，结果为 `47 pass / 0 fail`。

## 2026-04-22

### 新增

- 初始化酒店管理平台项目，采用前后端分离架构。
- 后端基于 Spring Boot、Spring Security、MyBatis-Plus、MySQL 构建接口服务。
- 前端基于 Vue 3 和 Vite 构建极简风格管理界面。
- 新增 JWT 登录鉴权和角色控制。
- 新增管理员、前台专员、住客三类角色。
- 新增管理员后台和前台工作台。
- 新增住客端入口，支持住客注册、登录、浏览房型、提交预订、查看我的订单和个人资料。
- 新增房型管理、房间管理、订单管理、住客管理、账户管理。
- 新增订单分页查询、条件筛选和关键词搜索。
- 新增订单状态流转：已预订、已入住、已退房、已取消。
- 新增入住登记、离店办理和取消订单。
- 新增订单自动计价，按房型单价和入住晚数计算房费。
- 新增费用明细拆分：房费、早餐、加床、押金、优惠券。
- 新增住客历史入住记录和消费统计。
- 新增仪表盘运营总览与近 7 日趋势分析。
- 新增 Excel 报表导出。
- 新增订单打印单、入住单、退房结算单接口。
- 新增管理员账户注册、用户角色、修改密码能力。
- 新增默认演示账号：管理员、前台专员、住客端账号。
- 新增透明底酒店 Logo 和浏览器 favicon。
- 新增项目截图并接入 README。
